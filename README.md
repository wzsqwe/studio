# PhotoStudio Pro - 自由摄影师全流程经营管理系统

## 📋 项目简介

PhotoStudio Pro 是一款专为独立摄影师打造的轻量化经营管理工具，核心围绕摄影项目全流程管理、收支利润核算、经营数据分析展开，无缝集成 AI 智能助手，实现"说话就能完成所有录入、查询、统计操作"。

## ✨ 核心功能

### 📊 首页核心经营看板
- 8个核心数据卡片（本月收入/利润/完结项目/待回款等）
- 近6个月收入&利润趋势折线图
- 项目类型收入占比饼图
- 智能提醒栏（逾期回款/档期/开票/报税等）
- AI智能助手对话面板

### 📁 摄影项目全流程管理
- 项目全生命周期状态管理
- 客户信息自动同步
- 合同与回款节点跟踪（定金/中期款/尾款）
- 成本明细逐条录入
- 项目模板功能

### 💰 收支与成本精细化管理
- 收支流水台账
- 变动成本/固定成本双体系
- 器材折旧自动分摊
- 数据同步与导出

### 👥 客户管理系统
- 客户档案库
- 历史合作自动同步
- 客户跟进记录
- 黑名单功能

### 📅 档期与产能管理
- 档期日历（月/周视图）
- 档期饱和度统计
- 产能利用率分析

### 🧾 发票与报税辅助
- 发票管理台账
- 报税报表自动生成
- 截止日期提醒

### 📈 多维度统计报表
- 月度/季度/年度报表
- 盈利分析/收入结构/回款分析/成本分析
- 盈亏平衡点测算
- PDF/Excel导出

### 🤖 AI智能助手
- 自然语言项目录入
- 对话式增删改查
- 智能信息补全
- 经营分析建议

## 🛠️ 技术栈

- **前端框架**: React 18
- **桌面框架**: Electron 28
- **数据库**: SQLite (better-sqlite3)
- **图表库**: Chart.js + react-chartjs-2
- **路由**: React Router DOM 6
- **样式**: CSS Variables + Flexbox/Grid

## 📦 安装与运行

### 环境要求
- Node.js 18+
- npm 或 yarn
- macOS / Windows

### 安装步骤

```bash
# 1. 进入项目目录
cd photostudio-pro

# 2. 安装依赖
npm install

# 3. 开发模式运行
npm run dev
```

### 构建桌面应用

```bash
# macOS
npm run build:mac

# Windows
npm run build:win
```

构建完成的可执行文件将位于 `dist/` 目录。

## 💡 使用说明

### 首次使用

1. 启动应用后，在左侧导航栏可以看到所有功能模块
2. 首页会自动展示经营看板，所有数据实时更新
3. 点击右上角的"AI助手"按钮可展开对话面板

### 新建项目

1. 进入"项目管理"页面
2. 点击"新建项目"按钮
3. 填写项目信息，包括客户、合同金额、拍摄日期等
4. 添加收款节点和成本明细
5. 保存后自动更新经营看板数据

### AI助手使用

在首页底部的AI对话面板中，您可以：

- **录入项目**: "接了个商业广告，收入15000，成本4000"
- **记录收支**: "收到定金5000" / "付了达芬奇年费268"
- **查询统计**: "这个月收入多少" / "还有多少没回款"
- **设置提醒**: "提醒我下周一给XX项目开票"

## 🔒 数据安全

- 所有数据存储在本地 SQLite 数据库
- 支持一键备份与恢复功能
- 可设置自动备份周期
