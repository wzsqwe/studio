# 自由摄影师全流程经营管理系统

## 项目概述
- **项目名称**: PhotoStudio Pro - 摄影师经营管理系统
- **项目类型**: Electron 桌面应用
- **核心技术栈**: Electron + React + SQLite + Chart.js
- **目标用户**: 个人自由职业摄影师

## 功能模块架构

### 1. 首页核心经营看板
- 8个核心数据卡片（实时联动）
- 近6个月收入&利润趋势折线图
- 项目类型收入占比饼图
- 智能提醒栏（逾期回款/档期/开票/报税/器材保养）
- AI智能助手对话面板（固定位置，可折叠）

### 2. 摄影项目全流程管理
- 项目全生命周期状态管理
- 客户信息自动同步
- 合同与回款节点跟踪
- 成本明细逐条录入
- 项目模板功能
- 操作留痕可查

### 3. 收支与成本精细化管理
- 收支流水台账
- 变动成本/固定成本双体系
- 器材折旧自动分摊
- 数据同步与导出

### 4. 客户管理系统
- 客户档案库
- 历史合作自动同步
- 客户跟进记录
- 黑名单功能

### 5. 档期与产能管理
- 档期日历（月/周视图）
- 档期饱和度统计
- 产能利用率分析

### 6. 发票与报税辅助
- 发票管理台账
- 报税报表自动生成
- 截止日期提醒

### 7. 多维度统计报表
- 月度/季度/年度报表
- 盈利分析/收入结构/回款分析/成本分析
- 盈亏平衡点测算
- PDF/Excel导出

### 8. AI智能Agent对话系统
- 自然语言项目录入
- 对话式增删改查
- 智能信息补全
- 经营分析建议

## 数据库设计

### 核心表结构
- `projects` - 项目表
- `clients` - 客户表
- `transactions` - 收支流水表
- `costs` - 成本明细表
- `invoices` - 发票表
- `schedules` - 档期表
- `equipment` - 器材表
- `reminders` - 提醒表
- `fixed_costs` - 固定成本配置表

## 界面风格
- 主题色：深蓝主色 + 金色点缀
- 卡片式设计
- 圆角 + 阴影
- 简约图标
- 响应式布局
